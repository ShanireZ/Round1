Round1 is a youth informatics practice platform for CSP-J/S and GESP 1-8. Core flow is offline content generation -> validate -> admin import/publish -> student attempts. Tech stack: TypeScript monorepo on Windows, Express backend, React/Vite client, offline scripts under scripts/, persistent question bundles under papers/, reports under artifacts/reports/, temporary draft/probe assets under artifacts/tmp/. Key standards live in standard/, especially offline content artifacts, agent/tool calling, domain/content, naming/structure.
