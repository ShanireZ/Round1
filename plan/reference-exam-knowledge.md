## 二、GESP 考试概览

### 2.1 考试定位

GESP（CCF 编程能力等级认证）共 8 级，每季度举办一次（3/6/9/12 月），C++ 和 Python 两个语种。本平台仅涉及 C++ 方向。

### 2.2 GESP 考试题型结构

| Section  | 题型   | 题数 | 分值/小题 | 满分    |
| -------- | ------ | ---- | --------- | ------- |

| 二       | 判断题 | 10   | 2 分      | 20      |
| 三       | 编程题 | 2    | 25 分     | 50      |
| **合计** |        |      |           | **100** |

> **考试时长**：120 分钟
>
> **平台适配说明**：本平台以"客观题模拟练习"为核心定位，GESP 试卷在平台上统一复用 CSP 初赛三段式结构（单选 + 阅读程序 + 完善程序），时长 60 分钟。知识点范围与难度严格对齐对应 GESP 级别。

## 四、GESP 各级 C++ 知识点范围（官方大纲详细版）

> 以下内容摘自 **CCF GESP C++ 认证标准** 官方 PDF，按级别列出知识点详述与知识块。
> 考试时长：1~4 级 120 分钟，5~8 级 180 分钟。C++ 标准：C++11（如无特殊说明）。

### GESP 一级

**考核目标**：掌握顺序、循环、分支的简单程序结构，可以使用集成开发环境进行编程与调试，完成单一功能的程序设计。

| 编号 | 知识块         | 知识点                                                               |
| ---- | -------------- | -------------------------------------------------------------------- |
| 1    | 计算机基础知识 | 计算机的软硬件组成（CPU、内存、I/O 设备等）、常见操作、发展历程      |
| 2    | 集成开发环境   | 创建文件、编辑文件、保存文件、编译、解释、调试（如 Dev C++）         |
| 3    | 结构化程序设计 | 顺序结构、分支结构、循环结构                                         |
| 4    | 程序的基本语句 | cin/scanf/cout/printf/赋值语句/复合语句/if/switch/for/while/do-while |
| 5    | 程序的基本概念 | 标识符、关键字、常量、变量、表达式的概念；程序的注释                 |
| 6    | 基本运算       | 算术运算、逻辑运算（&&/\|\|/!）、关系运算、变量自增与自减、三目运算  |
| 7    | 基本数据类型   | int, long long, float, double, char, bool                            |

**知识点详述**：
1. 了解计算机的基本构成，了解 Windows/Linux 等操作系统基本概念
2. 熟悉集成开发环境使用：创建、编辑、保存、编译、调试
3. 掌握基础 cin/scanf/cout/printf 与赋值语句
4. 掌握标识符、关键字、常量、变量、表达式的概念
